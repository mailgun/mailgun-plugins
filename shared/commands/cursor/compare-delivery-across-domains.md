# Compare Delivery Across Domains

Ask for the domain list, region, date range, and comparison goal.

Use consistent read-only analytics queries across all domains. Compare delivery rate, failure rate, complaint rate, unsubscribe rate, open/click engagement, provider/device/country splits, and suppression impact when available.

Return a ranked summary, a metrics table, caveats about unequal configuration, and recommended follow-ups.
