# Draft a Test Email Template

Ask for purpose, audience, sender domain, template name, subject, personalization variables, and whether the user wants HTML, text, or both.

Draft the content first. Recommend Validate for recipients and Inspect preview QA for HTML before sending or publishing. Confirm before creating template versions, preview tests, or outbound email.
